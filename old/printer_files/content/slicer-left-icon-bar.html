Interface: Left Icon Bar

Over the next several pages, we will describe PrusaSlicer’s user interface (UI). Each page will introduce one part of the program’s UI and its most important aspects.

We will start with the icon bar on the left side of the window. This houses various tools used for manipulating your imported objects. To use any of these tools, you must select an object first.

Move

By dragging the little boxes, you can move the model in each of the three axes. You can also drag the object directly with your mouse by pressing and holding the left mouse button.

When you pull any part of the model out of the boundaries of the print volume, it will turn blue. PrusaSlicer will not allow you to slice an object while it is blue.

Also, note that you can't leave the model hanging in the air; it will just drop back onto the plate. However, you can pull it under the plate - this is called Sinking. The program will then ignore the part under the plate and only slice the part above. This works as a simple cutting tool. The effect is then noted under that particular model as "Sinking". To put the object back on the plate, click the icon next to the Position values in the Object manipulation panel.

Scale

Drag the boxes to change the object's size . Use the orange boxes in the corners to maintain the same proportions. The other boxes will stretch or squeeze the model non-uniformly.

Rotate

Click and drag the red, green, or blue blocks to rotate the model. Move your mouse along white the guide lines to snap to fixed increments or you can freeley rotate the model by dragging outside of the lines.

Place on Face

Choose which surface you would like to align with the print bed, and the object will automatically snap to that orientation.

Other Tools

You are encouraged to research and experiment with the other tools found in this panel. However, they will not be covered as part of this training.
