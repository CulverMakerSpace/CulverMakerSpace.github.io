Initial Setup

The first time you launch PrusaSlicer on a new computer, you will need to complete an initial setup. This process only needs to be done once per machine.

After the setup is complete, the slicer will automatically reload your saved settings each time you open it. However, if you use a different computer in the future, you will need to go through the setup process again on that device.

You do not need to create an account or login, just click next on this screen.

Only select Prusa FFF from the online sources screen.

Scroll down to the MK4 Family section and select the “Original Prusa MK4S: 0.4 mm nozzle”. Note - Do not select the HF 0.4 mm nozzle!

Do not make any changes to this screen, just click next.

On the filament selection screen you first need to click the None button.

Now select (All) > PLA > Overture > Overture PLA

Do not make any changes to this screen, just click next.

Do not make any changes to this screen, just click next.

Do not make any changes to this screen, just click next.

Click the two boxes to associate both .3mf files and .stl files to PrusaSlicer

Change the view mode to Advanced.

Now you can click Finish to complete the setup process!
